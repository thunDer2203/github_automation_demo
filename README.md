# GitHub Automation Demo Repository

This repository is provided as a demo repository for testing the **GitHub Automation Bot**.

## Purpose

Use this repository to verify that the automation platform correctly responds to GitHub events such as:

- Opening a new Issue
- Opening a Pull Request

Depending on the configured automation rules, the application may:

- Automatically add labels
- Post comments
- Send Slack notifications
- Record events in the automation dashboard

## How to Test

1. Fork this repository (recommended), or use your own repository.
2. Connect the repository to the GitHub Automation application.
3. Create one or more automation rules.
4. Open a new Issue or Pull Request.
5. Verify that the configured actions execute successfully.
